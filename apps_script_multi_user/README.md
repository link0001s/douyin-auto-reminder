# 多用户并发版（免费、电脑关机可运行）

这套方案基于 `Google Apps Script + Google Sheets + Gmail`：

- 多个用户可以同时使用同一个网页
- 每个用户独立一条监控记录（独立抖音链接、独立违约通知邮箱）
- 定时任务在云端跑，电脑关机不影响

---

## 1）部署（一次性）

1. 打开：<https://script.new>
2. 新建两个文件：
   - `Code.gs`（粘贴本目录 `Code.gs` 全部内容）
   - `Index.html`（粘贴本目录 `Index.html` 全部内容）
3. 点击保存（项目名随便）
4. 先运行一次 `setupDailyTrigger`
   - 首次会弹授权，全部点允许
   - 成功后会创建“每天自动检查”触发器（09:10，上海时区）
5. 点击 `部署 -> 新建部署 -> 网页应用`
   - 执行身份：`我`
   - 访问权限：`任何人`
   - 部署后复制网页链接（这个链接可以发给所有用户）

---

## 2）用户怎么用

用户打开网页后：

1. 填抖音号/主页链接
2. 填违约通知邮箱
3. 选择规则周期（7/15/30）
4. 点 `保存状态`
5. 后续可随时点 `点检测 / 刷新状态`

系统会生成一个监控编号（`MON-...`），后续刷新状态要用这个编号。

---

## 3）邮件从哪里发

违约邮件由你部署脚本的 Google 账号发出（Gmail 通道，免费）。

可手动测试：

- 运行函数：`sendTestMailNow("2879154754@qq.com")`

---

## 4）注意事项

1. Gmail 免费额度有日发送上限（普通账号约 100 收件人/天）。
2. 抖音抓取可能受风控影响，失败会记录到“状态”和日志。
3. 这套是“多用户并发”设计，数据写入 Google Sheets 并使用锁防止并发冲突。
